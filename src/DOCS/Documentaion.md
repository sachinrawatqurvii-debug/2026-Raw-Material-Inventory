****************************************** transactions category **********************

Stock ko add ya ship karne ke liye Add/Ship / Fabric par click kare 
ab ek page open hoga wanha apne requirement ke according select operation type se ek operation choose karo [Add Stock , Ship Stock , Add Stock Quantity]

1. Add Stock ko select karne par 
(i) fabric number enter karo 
(ii) fabric source select karo
(iii) quantity enter karo meter me 
(iv) location enter karo 
(v) Add Stock button ko clik karo agar stock add ho jaayega to right side me popup aayega ki stock add ho chuka hai successfully


2. Ship Stock ko select karne par 
(i) Fabric number enter karo
(ii) stock ki quantity enter  karo meter me 
(iii) current location select karo stock ki (agar stock ko store 1 se ship karna hai to store1 select karo otherwise vender select karo)
(iv) ab destination select karo agar stock ko store 2 me shift karna hai to store2 select karo otherwise production select karo 
(v) Ship Stock button par click karo



3. Add stock quantity select karne par (ye option tab select kare jab kisi existing stock me stock ki quantity add karni ho)

(i) fabric number enter karo
(ii) stock ki quantity enter karo meter me 

Note : abhi quantity meter me add karo jaise hi changes aa jayega kg me quantity enter karna wo automatically meter me convert hokar stock me add ho jaayega


Accessory ko update karne ke liye Accessory Update tab par click kare

(1) Accessory number enter karke search kare 
(ii) Action type se add stock ship stock ko apne requirement ke according pick kare aur 
(iii) aur stock ki quantity unit me enter kare jaise ki (1,2,3 etc)
(iv) Aur jo option select kiya hai wo action button aayega agar add stock kiya hai to add unit button ko click karo otherwise ship units button ko click karo
(v) Accessory update ho jaayegi

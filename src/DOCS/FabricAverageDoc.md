# FabricTable Component -- User Guide (README)

## Overview

`FabricTable` is a React component that displays detailed average
measurements for fabric patterns, fabrics, and accessories.\
It includes:

-   API data fetching\
-   Loading & error states\
-   Search functionality\
-   Sorting\
-   Expandable rows\
-   Statistics summary cards\
-   Stylish responsive UI (TailwindCSS)

------------------------------------------------------------------------

## 📌 API Used

The component fetches data from:

    GET https://raw-material-backend.onrender.com/api/v1/average

Expected Response Structure:

``` json
{
  "data": [
    {
      "patternNumber": "562",
      "style_number": 12065,
      "styleImage": "img-url",
      "fabrics": [
        {
          "average_xxs_xs": 1.2,
          "average_s_m": 1.4,
          "average_l_xl": 1.5,
          "average_2xl_3xl": 1.5,
          "average_4xl_5xl": 2,
          "width": "Normal"
        }
      ],
    }
  ]
}
```

------------------------------------------------------------------------

## 🎯 Features

### ✔ Fetch Data Automatically

Runs once on mount using `useEffect()`.

### ✔ Loading State

Shows animated spinner until API responds.

### ✔ Error Handling

Displays retry button if API fails.

------------------------------------------------------------------------

## 🔍 Search Functionality

You can search by:

-   Pattern Number\
-   Fabric Width\
-   Accessory Measurements

Search is case-insensitive.

------------------------------------------------------------------------

## ↕ Sorting

Sorting is available for:

-   `patternNumber`
-   `style_number`

Sort icon toggles between ASC/DESC using:

``` js
handleSort()
```

------------------------------------------------------------------------

## 🔽 Expandable Rows

If a pattern contains multiple fabrics, users can expand to view all.

``` js
toggleRowExpand(index)
```

------------------------------------------------------------------------

## 🧾 Table Columns

  Column        Description
  ------------- ------------------------------------------
  Style No.     `style_number`
  Pattern \#    `patternNumber`
  Fabrics       Shows fabric averages (XXS--5XL + Width)
  Accessories   Shows average values by size range
  Actions       Expand / Collapse row


# Stock2 Component Documentation

## Overview
`Stock2` is a React component designed to display and manage fabric stock inventory. It supports:

- Search by fabric number, name, or style number  
- Pagination  
- Expandable rows to display style numbers  
- Dynamic stock color indicators  
- Smooth scrolling  
- Loading and empty states  

---

## Features

### 🔍 Search Functionality
Users can search using:
- Fabric Number  
- Fabric Name  
- Style Numbers  

Search activates when pressing **Enter**.

---

### 📄 Pagination
- Displays **50 items per page**  
- Provides **Next**, **Previous**, and direct page selection  

---

### 🔽 Expandable Style Numbers
Each row has a button that toggles the visibility of its linked style numbers.

---

### 🟢🟡🔴 Stock Color Indicator
Based on `availableStock`:

| Stock Value | Color |
|-------------|--------|
| > 100       | Green  |
| > 50        | Yellow |
| ≤ 50        | Red    |

---

### 📦 Props from Context
`useGlobalContext()` provides:
- `stock2` → inventory list  
- `stockLoading` → loading state

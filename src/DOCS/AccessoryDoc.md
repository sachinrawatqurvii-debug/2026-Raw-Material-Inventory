# AccessoryList Component README

## Overview

The `AccessoryList` React component provides a fully functional UI for
displaying, searching, and paginating accessory inventory records
fetched from a backend API.

It includes: - Search functionality\
- Pagination\
- Loading state\
- Summary cards\
- Styled table view

------------------------------------------------------------------------

## API Endpoint

The component fetches data from:

    https://raw-material-backend.onrender.com/api/v1/accessory/

### Query Parameters:

  Param      Description
  ---------- ----------------------------
  `page`     Current page number
  `limit`    Number of records per page
  `search`   Search keyword

------------------------------------------------------------------------

## Main Features

### ✔ Fetch Accessories

Data is fetched whenever `page` or `search` changes using:

``` js
useEffect(() => {
    fetchAccessories();
}, [page, search]);
```

### ✔ Controlled Search Box

Updates search and resets page:

``` js
onChange={(e) => {
    setSearch(e.target.value);
    setPage(1);
}}
```

### ✔ Pagination System

-   Previous / Next navigation\
-   Dynamic page buttons (max 5 at a time)\
-   Total pages calculated by:

``` js
Math.ceil(total / limit)
```

## Data Rendering

Each accessory row displays: - Serial Number\
- Style Number\
- Accessory Number\
- Accessory Name\
- Type\
- Stock Unit (styled with color badges)\
- Created Date & Time

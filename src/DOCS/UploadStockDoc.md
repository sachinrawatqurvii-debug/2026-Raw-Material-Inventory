# StockUploaderCSV - Technical Documentation

## Overview

`StockUploaderCSV` is a React component designed for uploading CSV files
containing fabric stock data. It parses CSV using **PapaParse**,
transforms rows into backend-friendly JSON, and sends the data to the
Raw Material Backend API. It also includes UI features like a progress
bar, error handling, and success summaries.

------------------------------------------------------------------------

## Features

-   CSV parsing using **PapaParse**
-   Auto-progress simulation
-   Upload status indicators
-   Error and success message UI
-   CSV template download
-   File cancel/reset functionality
-   Supports multiple style numbers per fabric
-   Clean, modern TailwindCSS UI

------------------------------------------------------------------------

## Expected CSV Format

The CSV must contain the following columns:

  Column Name     Description
  --------------- -------------------------------
  `Fabric Name`   Name of the fabric
  `Fabric #`      Unique numeric identifier
  `Style #'s`     Comma-separated style numbers

Example:

    Fabric Name,Fabric #,Style #'s
    Cotton,101,201,202
    Linen,102,300,301

### Data Transformation

Each CSV row → JSON payload:

``` json
{
  "fabricNumber": 101,
  "fabricName": "Cotton",
  "styleNumbers": [201, 202]
}
```

### API Endpoint

    POST /api/v1/stock/create
